# Shakespeare-Sonnets

In the JSON, under the "sonnets" object, each individual sonnet has the following:
- "number": The sonnet number according to the original quarto publication;
- "modern": A modern edited version from the [Folger Shakespeare Library](https://www.folger.edu/explore/shakespeares-works/shakespeares-sonnets/read/);
- "quarto": The original spelling from the 1609 quarto, taken from the University of Victoria's [Internet Shakespeare Editions](https://internetshakespeare.uvic.ca/doc/Son_Q1/complete/index.html);
- "first_line": The first line of each sonnet from the Folger edition

Notes:
- The Folger edition uses "-èd" when that ending is pronounced when it normally isn't (eg. blessèd as "bleh-sehd" instead of "blehsd"; or usèd as "yoo-zehd" instead of "yoozd") "and "-ed" if it is not pronounced or when it normally is (eg. blessed as "blehsd" instead of "bleh-sehd"; or ragged not being spelt as raggèd since it is normally pronounced "ra-gehd".)
- While the sonnets normally have 14 lines each, Sonnet 99 has 15 lines, and Sonnet 126 has 12 lines.
- Some readings in the modern version may be substantially different from the original publication (eg. Sonnet 146, line 2). [See here](https://www.folger.edu/explore/shakespeares-works/shakespeares-sonnets/textual-notes/) for Folger's editing choices.
